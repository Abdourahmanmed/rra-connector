# rra-connector

Monorepo for the RRA fiscal connector between Sage 100 Gestion Commerciale and RRA through VSDC.

## Structure

- `apps/api`: Backend API (Node.js, TypeScript, Express, Prisma)
- `apps/web`: Frontend app (Next.js App Router)
- `packages/shared`: Shared types/constants
- `prisma`: Prisma schema
